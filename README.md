# Infotafel-Zoo

## Umschulung 2022 FI-AE - LuG Academy
### Aufgabe

Ein kleiner Zoo benötigt ein Programm für einen automatisierten
Infostand.

Auf ihm soll man sich die Beschreibung der verschiedenen Tiere 
anzeigen lassen können, ebenso wie den Weg zum jeweiligen Gehege.

Der Anwender soll ein Menü zur Auswahl haben, womit er zuerst das
Tier wählen kann und anschliessend, ob er eine Wegbeschreibung
zum Gehege benötigt oder ob er eine Beschreibung des Tiers haben
möchte.

1)	Entwickeln Sie die Klasse "Tier" mit allen ihrer Meinung
	nach benötigten Feldern und einem vollständigen Konstruktor.

2)	Erstellen Sie 5 verschiedene Tiere mit allen Feldern.

3)	Erstellen Sie ein Menü (gerne auch als statische Methode,
	dies ist aber kein muss) mit dem der Anwender das Tier
	wählen kann.

4)	Nach der Wahl des Tieres, soll der Anwender nun entscheiden,
	ob er eine Beschreibung oder eine Wegbeschreibung wünscht.

5)	Je nach Auswahl soll nun eine entsprechende Ausgabe erscheinen:
	Beschreibung oder Wegbeschreibung.
